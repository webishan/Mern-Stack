# Notes

## `list.html`
- Demonstrates a horizontal navigation menu using an unordered list.
- Uses `display: inline-block` on list items.
- Styles anchor tags with padding, tomato background, white text, and hover effects.
- Includes a `div` with a `clip-path` polygon to create a custom shape.

## `units.html`
- Demonstrates CSS units and basic inheritance concepts.
- Contains comments showing examples of absolute units and relative units.
- Resets default margin and padding with the universal selector.
- Sets a full-screen `div` using `100vw` and `100vh`.

## Topics covered
- CSS list styling
- Navigation menu layout
- Hover transitions and shadows
- Clip-path shapes
- CSS units: px, em, rem, vw, vh, vmin, vmax, percent
