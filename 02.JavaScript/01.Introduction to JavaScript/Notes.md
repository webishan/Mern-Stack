
## Key Points
- JavaScript is a programming language used to make web pages interactive.
- It runs in the browser and can also run outside the browser with tools like Node.js.
- It is used for things like buttons, forms, animations, APIs, and full web applications.
- JavaScript is a high-level, interpreted language, which makes it easier to start learning.
- Variables store data, and you can create them with `let`, `const`, or `var`.
- `const` is used when the value should not be reassigned, while `let` is used when the value may change.
- Functions let you group code and reuse it whenever needed.
- JavaScript can respond to events such as clicks, typing, and page loading.

## Simple Example
```javascript
let name = "JavaScript";

function greet() {
	console.log("Hello from " + name);
}

greet();
```

This example shows a variable, a function, and output in the console.

## Why It Matters
- JavaScript is one of the core technologies of the web, along with HTML and CSS.
- HTML gives structure, CSS gives style, and JavaScript gives behavior.
- Learning JavaScript is important if you want to build modern websites or web apps.
