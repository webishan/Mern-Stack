CSS Basics

A hands-on reference covering the core concepts of CSS

1. Selectors

CSS selectors target HTML elements so you can apply styles to them.

Element — targets every <p> tag.
Class — targets elements with class="highlight" (prefixed with .).
ID — targets the unique element with id="page-header" (prefixed with #).
Descendant — .card p targets every <p> inside a .card.
Child — .section > h2 targets only direct <h2> children.

2. The Box Model

Every element is a rectangular box made of four layers (from inside out): content → padding → border → margin.

Content
padding: 16px
border: 4px solid

Setting box-sizing: border-box makes width and height include padding and border, which is the most predictable behaviour.

5. Flexbox

Flexbox arranges items along a single axis (row or column) and distributes space between them automatically.

Item 1
flex: 1 1 140px
Item 2
flex: 1 1 140px
Item 3
flex: 1 1 140px
Item 4
flex: 1 1 140px

Key properties: display: flex, justify-content, align-items, flex-wrap, gap.

6. CSS Grid

Grid places items in a two-dimensional layout using rows and columns.

Key properties: display: grid, grid-template-columns, gap, grid-column, grid-row.

7. Pseudo-classes & Pseudo-elements

Pseudo-classes style an element based on its state or position (:hover, :nth-child, :first-child). Pseudo-elements style a specific part of an element (::before, ::after, ::first-line).

  • Odd row — hover me
  • Even row — hover me
  • Odd row — hover me
  • Even row — hover me

The first item gets a ★ via ::before. Odd/even rows use :nth-child.

9. Responsive Design & Media Queries

Resize the browser window below 600 px — the grid collapses to a single column and the heading shrinks.

Media queries let you apply different styles depending on the viewport size, orientation, or other device characteristics: 

@media (max-width: 600px) {
  .grid-container {
    grid-template-columns: 1fr;
  }
}
